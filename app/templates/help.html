Chart Data

To create the chart data, there is an Excel file located in the app/data folder named "new_trade_log.xlsx".

Please enter your trades in each row. The key columns to fill in are:

  • Open Trade - Date, Quantity, Price, Amount spent for the trades
  • Close Trade - Date, Quantity, Price, Amount received after closing the trades

Other columns are explanatory and meant for potential future updates if needed.

Important

The opening balance is specified in the "crden.txt"file under the 4th line.

The Initial Amount indicated is 10000/-.

If you happen to add more funds later, please update it here.

Note Please esure that the amount should be written as a single figure. Do not use expressions like 10000+500, but instead mention it as a total, for example, 10500.

Abount Exch Utility

Here, we have three sections for Exchange-related activities:

1. Last Business Dates

Here, we add business dates to our calendar. It has been already updated until the last business date mentioned in the box.

Please mention the dates when the exchange opens. The format is as follows: DD-MM-YYYY (example 01/12/2024).

Note :

These calendar dates are used to fetch data from the exchange and update/add to our database.

Missing any dates means we won't have data for those dates.

2. Data Download

Enter dates for which data is to be downloaded from the Exchange & Calendar have dates.

Note :

 

Caution: Do not input dates exceeding a 15-day range, as this action may result in your IP address being blocked.

 

3. Database Update

Enter dates for which the database needs to be updated with the data downloaded from the Exchange & Calendar have dates.
